Lezione 3 — Creare una pagina con JavaScript

Ricreare il vostro progetto dinamicamente con JavaScript partendo da un HTML vuoto.

Procedimento

1. Creare un tag HTML da aggiungere alla pagina

Per creare un tag HTML da aggiungere alla pagina, occorre usare il metodo document.createElement().

Esempio — creare un paragrafo <p>

const nuovoParagrafo = document.createElement("p");
  • const è una parola chiave per dichiarare una variabile costante.
  • nuovoParagrafo è il nome della variabile che conterrà il nuovo elemento.
  • document.createElement è il metodo che crea un nuovo elemento HTML.
  • ("p") è il tipo di elemento da creare, in questo caso un paragrafo. Notare i doppi apici attorno al nome dell'elemento.
nuovoParagrafo.textContent = "Questo è il testo del nuovo paragrafo.";
  • nuovoParagrafo è la variabile che contiene il nuovo elemento creato.
  • textContent è una proprietà che imposta il testo del nuovo elemento.
  • "Questo è il testo del nuovo paragrafo." è il testo che verrà visualizzato nel paragrafo.

2. Stilizzare l'elemento creato

Per stilizzare un elemento prima di aggiungerlo alla pagina occorre usare classList.add()

Tra le parententesi si mette il nome di una classe CSS che abbiamo creato nel nostro CSS

Esempio — aggiungere una classe al paragrafo creato

nuovoParagrafo.classList.add( "mia-classe");

3. Aggiungere l'elemento creato alla pagina

Per aggiungere un elemento alla pagina occorre usare il metodo appendChild()

Occorre prima però selezionare un elemento già esistente a cui aggiungerlo.

Esempio — aggiungere un paragrafo a un <div>

Prima seleziono il div a cui aggiungere il paragrafo

const divGiaEsistente = document.getElementById("div-gia-esistente");

Poi aggiungo effettivamente il paragrafo

divGiaEsistente.appendChild(nuovoParagrafo);

NOTA per creare un div che contiene degli p, occorre:

  1. Creare il div con document.createElement("div")
  2. Creare un p con document.createElement("p")
  3. Impostare il testo del p con p.textContent = "Testo del paragrafo"
  4. Eseguire i passaggi 2 e 3 per ogni p da aggiungere
  5. Aggiungere il div alla pagina con document.body.appendChild(div)
